# Document Archive
### Simple Document Archive with Blockchain based Integrity

Create *application.properties* in *src/main/resources* containing:
```java
user_name=

# E-mail Support
email_username=
email_password=
email_smtp=
email_recipient=

# Twitter Support
twitter_consumerKey=
twitter_consumerSecret=
twitter_accessToken=
twitter_accessTokenSecret=
```
